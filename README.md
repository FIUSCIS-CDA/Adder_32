#### MODULE NAME
       Adder_32

#### DEPENDENCIES
       OneBitAdder
       Grounder

#### INPUTS

Name | Size(Bits)
---------|---------
A   |     32    
B   |     32

#### OUTPUTS

Name | Size(Bits)
---------|---------
S   |     32

#### DESCRIPTION
       Computes  A  +  B,  where  A and B are 32-bit values.  Stores result in
       32-bit value S.
